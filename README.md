# 🎓 BNMC College Website

A modern, responsive, and user-friendly college website built using **HTML, CSS, Bootstrap, and JavaScript**. This project showcases the official website design for **Badri Narayan Mukteshwar College (BNMC)** with multiple pages, responsive navigation, and a clean user interface.

## 🚀 Live Demo

🔗 **Website:** https://itssameer666.github.io/bnmc/

## 📂 GitHub Repository

🔗 **Repository:** https://github.com/Itssameer666/bnmc

## ✨ Features

- 🏫 Home Page
- 📖 About College
- 👨‍🏫 Principal's Message
- 🎓 Courses Section
- 📝 Admission Information
- 📞 Contact Page
- 🖼️ Gallery
- 📊 Statistics Section
- 📱 Fully Responsive Design
- 🎨 Bootstrap 5 Components
- ✨ AOS Scroll Animations
- 🌐 Google Maps Integration
- ⬆️ Scroll-to-Top Button
- 📄 Multi-page Website

## 🛠️ Built With

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Font Awesome
- AOS (Animate On Scroll)

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Itssameer666/bnmc.git
```

Open the project folder and run **index.html** in your browser.
